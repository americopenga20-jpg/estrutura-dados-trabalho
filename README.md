## Descrição
Este projeto implementa e compara três algoritmos de ordenação:

- Bubble Sort
- Selection Sort
- Insertion Sort

O objetivo é compreender o funcionamento dos algoritmos e analisar a complexidade temporal usando notação Big-O.
