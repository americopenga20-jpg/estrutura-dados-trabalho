 Comparação de Algoritmos de Ordenação

Introdução
A ordenação de dados é fundamental para organizar informações e facilitar a busca.

 Algoritmos Estudados
- Bubble Sort
- Selection Sort
- Insertion Sort

 Implementação
Os algoritmos foram implementados em linguagem C.

 Resultados
Todos os algoritmos ordenam corretamente, porém apresentam diferentes desempenhos.

 Análise de Complexidade
- Bubble Sort → O(n²)
- Selection Sort → O(n²)
- Insertion Sort → O(n) a O(n²)

## Conclusão
A escolha do algoritmo depende do contexto e dos dados utilizados.
